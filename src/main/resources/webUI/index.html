Law Cases Tracking System

LCTS is a training project created in order to improve programming skills and create something useful.
The main idea is to help Ukrainian attorneys with tracking the appointed court hearings.
Using the open state register LCTS looks for and collects the information about your court cases which are assigned for court proceedings.
Just add the case numbers. LCTS will track them automatically.

Инструкции:

1. Чтобы добавить номер дела вставьте его в форму и нажмите кнопку Add number. Номер дела должен быть введен точно в таком же виде и формате, в каком он был сгенерирован в суде. Пример номера: 111/2232/17-ц

2. Новый поиск по делам проводится при каждой загрузке страницы / добавлении нового номера / удалении / нажатии на кнопку обновления. Если информация по делу по каким-то причинам не найдена, дело будет отображено в таблице с пометками 'no info'.

3. Для удаления дела нажмите кнопку с изображением корзинки в строке соответсвующего номера.

4. Чтобы добавить напоминание в Google Calendar - нажмите на дату в соответствующей строке.

Инофрмация о версии:
Текущая версия приложения поддерживает поиск в следующих судах: Приморский г. Одессы, Киевский г. Одессы, Суворовский г. Одессы и т. д.
Поиск дел в судах апелляционных инстанций в текущий момент недоступен.
